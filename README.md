## 📖 Enunciados

Abaixo, você encontra a descrição de cada questão implementada neste repositório:

<details>
<summary><b>🔹 Estruturas Condicionais e Repetição (Q1, Q8, Q19, Q20)</b></summary>

* **Q1 - Divisores:** Exibe divisores de um número positivo.
* **Q8 - Números Primos:** Verifica se um número informado é primo.
* **Q19 - Contador de Positivos:** Lê números até encontrar um negativo e exibe a contagem.
* **Q20 - Sistema de Login:** Simula acesso com bloqueio após 3 tentativas erradas.
</details>

<details>
<summary><b>🔹 Vetores e Cadeias de Texto (Q2, Q3, Q7, Q15, Q16)</b></summary>

* **Q2 - Contador de Vogais:** Conta vogais em uma frase usando bibliotecas de texto.
* **Q3 - Par ou Ímpar:** Separa 10 números em dois vetores distintos.
* **Q7 - Palíndromo:** Verifica se uma cadeia de caracteres é idêntica de trás para frente.
* **Q15 - Soma de Vetores:** Gera um terceiro vetor a partir da soma de outros dois.
* **Q16 - Negativos no Vetor:** Função que identifica a quantidade de números negativos.
</details>

<details>
<summary><b>🔹 Matrizes (Q6, Q13, Q18, Q21)</b></summary>

* **Q6 - Matriz Identidade:** Valida se uma matriz 3x3 é uma identidade.
* **Q13 - Maior e Menor:** Encontra os valores extremos em uma matriz 4x4.
* **Q18 - Diagonal Secundária:** Calcula a soma dos elementos da diagonal secundária (4x4).
* **Q21 - Maior Soma de Linha:** Identifica qual linha de uma matriz 5x5 possui a maior soma.
</details>

<details>
<summary><b>🔹 Funções e Recursividade (Q4, Q5, Q10, Q12, Q14)</b></summary>

* **Q4 - Potenciação:** Função manual de potência (sem bibliotecas prontas).
* **Q5 - Ordenação (Bubble Sort):** Algoritmo de ordenação crescente manual.
* **Q10 - Inversão de Cadeia:** Função **recursiva** para inverter textos.
* **Q12 - Fatorial:** Cálculo de fatorial utilizando **recursão**.
* **Q14 - Triângulo Numérico:** Gera uma pirâmide de altura N.
</details>

## 🚀 Como executar

1.  Baixe e instale o **Portugol Studio**.
2.  Clone este repositório: 
    ```bash
    git clone [https://github.com/](https://github.com/)[seu-usuario]/[seu-repositorio].git
    ```
3.  Abra o Portugol Studio e importe os arquivos `.por`.
4.  Pressione `F8` para executar os algoritmos.
